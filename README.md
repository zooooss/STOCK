# STOCK
Efficient inventory management for restaurants and businesses

**Stock** is a smart and efficient inventory management service for restaurants and small businesses.  
It replaces messy text messages, spreadsheets, and manual records with a streamlined, easy-to-use platform.

## Features

- ✅ Real-time inventory tracking and management  
- ✅ Quick and easy stock updates via web or messaging integrations  
- ✅ Supports multiple business types (restaurants, cafes, small stores)  
- ✅ Data-driven stock insights and notifications  

## Tech Stack

- **Frontend:** React  
- **Backend:** Node.js / Express  
- **Database:** MongoDB
